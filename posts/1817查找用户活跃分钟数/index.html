1817. 查找用户活跃分钟数

我的题解：

这道题主要是哈希表的知识点，但是我们可以用defaultdict来做，可以省下几行代码量。 defaultdict可以假设存在key的初始值，所以我们只用直接append()或者add()就行。 这次和官方题解一模一样呢!

class Solution:
    def findingUsersActiveMinutes(self, logs: List[List[int]], k: int) -> List[int]:
        dict1 = defaultdict(set)
        list1 = [0] * k
        for i, j in logs:
            dict1[i].add(j)
        for i in dict1:
            print(dict1[i])
            # TypeError object of type ‘type‘ has no len() 是因为dict少打了1
            list1[len(dict1[i]) - 1] += 1
        return list1
